

##  From a remote host to the local host ##
- scp your_username@remotehost.edu:/[directori]/[file] /some/local/directory
	> -r : for directori

## From the local host to a remote host  ##
- scp [resource] your_username@remotehost.edu:/some/remote/directory
	> -r : for directori

## Use other port
> -P [port]
