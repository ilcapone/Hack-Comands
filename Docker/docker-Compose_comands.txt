
## Build images ##
docker-compose build

	## Run rocker compose and rebuild images
	docker-compose up --build

## Run Attached mode ##
docker-compose up

## Run deatached mode ##
docker-compose up -d

## Container Runing ##
docker-compose ps

## View services ##
docker-compose ps

## View logs after deatach mode ##
docker-compose logs -f -t [servicename]

## Deatach from logs ##
Ctrl + z or Ctrl + c

## Removing services and run containers ##
docker-compose down

	## Delete also persistence volumens
	docker-compose down --volume

## Start exited containers ##
docker-compose start

## Restart services #
docker-compose restart

## Restart single contaienr #
docker-compose restart [docker]
